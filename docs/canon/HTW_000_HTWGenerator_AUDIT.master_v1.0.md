## 6. Major warnings

### A. Do not collapse semantics
Do not let:
- rails
- checker layers
- procedural overlays
- baked layers
- noise
be narrated as one unified “effects thing”.

That is false and dangerous.

### B. Do not touch Icon casually
Icon is not finished, but it is more stable than the rest.
Audit it carefully before scheduling big changes.

### C. Do not treat chats as canon
Closed docs and repo truth outrank old chat wording.

### D. Do not audit from stale authority
Older audits were written under older WRKOPS/registry references.
This file updates the reading for the current doc pack.

### E. Do not let UI density be “cleaned” into blandness
The forensic / tool-heavy vibe is part of product value.
