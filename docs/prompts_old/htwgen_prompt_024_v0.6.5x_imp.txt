C. Exact implementation steps

1. Create helper ABOVE sbEffects()

Add this function verbatim:

function renderLayerRow(i, layer) {
  return `<div class="tog" style="padding-left:4px" data-layer="${i}" data-layer-visible="${i}">
    <span class="togl">Layer ${i + 1}</span>
    <span class="togk" data-layer-delete="${i}" style="cursor:pointer; margin-right:6px">DEL</span>
    <div class="togp${layer.visible ? ' on' : ''}" data-layer-visible="${i}">${layer.visible ? 'ON' : 'OFF'}</div>
  </div>`;
}

CRITICAL:
- Copy EXACTLY
- ASCII only
- Do NOT retype manually if avoidable
- Do NOT change spacing inside attributes
- Do NOT touch class names or data-* attributes

2. Replace inline template in sbEffects()

BEFORE:

for (let i = 0; i < S.bakeStack.length; i++) {

AFTER:

let rows = '';

for (let i = 0; i < S.bakeStack.length; i++) {
  const layer = S.bakeStack[i];

  + '</div><div class="sbs"><div class="sbdiv"></div></div>';
